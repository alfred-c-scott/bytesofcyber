First Login

Once Installed navigate to the IP address you assigned for your LAN port during install

You will be greeted with the following screen



Changing the Theme

The first thing I am going to do is change the theme because I don't like the bright white background that it defaults to

At the top, find the System Dropdown and navigate from System -> General Setup -> Theme

Under webConfigurator find the Theme Dropdown and change the theme to pfSense - Dark

After this change the home screen will look like this



Changing the default password for system created user, admin

Notice in the above image the WARNING at the top of the page. We need to Change the default password for the user, admin, next

The user, admin, was created during installation. In addition to changing the password I am going to create a new admin user

Choose a strong password using a generator, I Use Bitwarden, but there are other options out there.

After Entering password you need to make sure that you are a member of the admins group. Under Group Membership, Select admins, and then click the >> Move to 'Member of' list

See the screenshot below, when you have selected the new user and entered a complex password click the save button at the bottom



After you click the save button you will be redirected to the url below, which can be found at System -> User Manager -> Users



Disable login for system created user, admin

The next thing I am going to do is disable GUI login for the admin user for a little added security. We changed to a strong password, but it would be much better if removed a known variable, the "admin" username, from the list of possible attack vectors

NOTE 1: logout and then login with the new user you just created before making the following change to the 'admin' account

NOTE 2: This step can only be completed if you have added your new user to the 'admins' group

Again this step is optional but if you want to do this click on the pencil/edit icon to the right of the list of users on the System -> User Manager -> Users page. And then check the radio button that says "This user cannot login", and then click the save button at the bottom



After completing the steps above you will be redirected to the following page, Note the change in the Status column of the admin user
